## Reporting a Vulnerability

We take the security of X-Gate seriously. If you have found a security vulnerability in X-Gate, please report it to us.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to the repository maintainer. We will review your report and respond as soon as possible.
